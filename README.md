# Od. Diana Blanco — Dental Clinic Landing Page

Professional landing page built for a dental clinic based in Buenos Aires, Argentina.

## 🔗 Live Site

[od-dianablanco.com.ar](https://www.od-dianablanco.com.ar/)

## 📋 About

A clean, mobile-first landing page designed to present dental services,
location, and contact information. Patients can book appointments directly
via WhatsApp integration.

## ✨ Features

- Mobile-first responsive design
- Services showcase (Orthodontics, Pediatric Dentistry, Oral Health)
- WhatsApp appointment booking integration
- Location section with address details
- SEO optimized with sitemap.xml and robots.txt

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
